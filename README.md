# ParseHelper

ParseHelper 是一个基于 Chrome / Edge 浏览器扩展的 B 站视频解析助手。它可以在 B 站首页、搜索结果页、视频页等页面中，通过右键菜单快速提取视频的 BV 信息和分 P 信息，并生成可用于解析服务的链接，复制到剪贴板后便于在其他播放器或平台中使用。

## 功能特点

- 在 B 站页面中右键点击视频内容时，自动识别当前视频信息
- 支持从页面状态、URL、DOM 元素中提取 BV 和分 P 数据
- 提供右键菜单「复制B站解析链接」
- 可通过弹窗或设置页自定义解析服务器地址
- 支持复制成功后的提示反馈

## 项目结构

- manifest.json：扩展清单，定义权限、后台脚本、内容脚本和页面入口
- background.js：负责右键菜单、复制逻辑、解析地址构造和设置存储
- content.js：负责检测当前页面的视频信息，并把信息暴露给后台脚本
- popup.html / popup.js：扩展弹窗界面，用于快速修改解析地址
- options.html / options.js：扩展设置页，提供更完整的配置入口

## 安装方法

1. 打开浏览器扩展管理页：
   - Chrome / Edge：访问 chrome://extensions
2. 开启「开发者模式」
3. 点击「加载已解压的扩展程序」
4. 选择当前项目目录中的 ParseHelper 文件夹
5. 扩展安装完成后，点击扩展图标或进入设置页配置解析服务器地址

## 配置解析服务器

扩展默认使用以下占位地址：

```text
https://your-worker-domain.workers.dev/parse?url=
```

你需要将其替换为自己的解析服务地址。解析服务应当支持接收以下参数：

- url：视频 BV 号，例如 `BV1...`
- p：分 P 编号，例如 `2`

示例：

```text
https://your-worker-domain.workers.dev/parse?url=
```

服务端应当根据 `url` 和 `p` 参数返回视频直链或进行重定向。

## 使用方法

1. 打开 B 站页面并进入视频内容
2. 右键点击视频封面、标题或页面中的相关视频元素
3. 选择「复制B站解析链接」
4. 解析链接会被复制到剪贴板，随后可以直接粘贴到目标播放器或其他工具中使用

## 注意事项

- 当前扩展仅对 B 站域名生效
- 右键菜单依赖页面中的视频信息识别，某些特殊页面可能需要手动点击视频相关元素
- 若解析服务不可用，请先确认服务器地址配置正确

## 适用场景

- 需要将 B 站视频链接转成可解析的直链
- 在 VRChat、直播/视频播放器或其他第三方环境中使用 B 站视频内容
- 希望快速从浏览器中复制解析地址而不需要手动拼接链接
